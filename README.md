# dhtc

dht crawler with a web ui<br>

- [X] Counter of found torrents
- [X] Search by
  - [X] Name
  - [X] Info hash
  - [X] File name
  - [X] Date (only by day)
- [X] Search types (does not work for Date category)
  - [X] contains
  - [X] equals
  - [X] starts with
  - [X] ends with
- [ ] Statistics
  - [ ] Line charts for day, week, month, year
  - [ ] Pie / Bubble charts for categories

## how to..
### ..run locally
```shell
go run main.go
```
### ..run containerized
```shell
./run.sh
```
